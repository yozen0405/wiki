305 . 括號國

Description

括號國是一個奇怪的國家，那裡的人民非常喜歡左小括號 ( 以及右小括號 )，因此兩個括號國的人民很常使用括號語作為溝通的方式。舉例來說：

「))(()()」

「)()(()()()」

上述呈現了兩個人的對話，其中一個人在膜拜，另一個人在裝弱。

在那裡的競程選手也喜歡使用括號語來表示一道題目，像是：

「)()))(()())))(()())))()()()))()()))()((()))()))))((())」

經由翻譯後，這道題目的意思是：

「給定一個由 () 組成的字串，請問它所有子字串的最長合法括號子序列長度總和為何？」

你不想輸給括號國的競程選手，因此請想辦法解決這個問題。

一個括號字串為合法括號若它滿足以下條件：

  • 該字串的 ( 與 ) 出現次數相同

  • 對於該字串的任意前綴，其 ( 的數量皆不小於 )

一個括號字串 s1s2,…,sN 的最長合法括號子序列為最長的序列 a1<a2<…<aK 使 sa1sa2,…,saK 是一個合法括號。

一個長度為 N 的字串有 (N+12) 個子字串，對於任意 1≤l≤r≤N，slsl+1,…,sr 皆為字串 s 的子字串。

Input Format

輸入第一行有一個整數 N，代表括號字串的長度。

輸入第二行有一個由 ( 與 ) 組成的字串 S。

  • 1≤N≤2×105

  • |S|=N

  • Si∈{(,)}

Output Format

請輸出一行，上面有一個數字代表答案。

Sample Input 1

4
()()

Sample Output 1

12

8
)))(((((

Sample Output 2

Sample Input 3

)(()))(

Sample Output 3

Hints

令 f(l,r) 為 SlSl+1,…,Sr 的最長合法括號子序列長度，以下為範例測資一中所有可能的函數值：

  • f(1,1)=0

  • f(2,2)=0

  • f(3,3)=0

  • f(4,4)=0

  • f(1,2)=2

  • f(2,3)=0

  • f(3,4)=2

  • f(1,3)=2

  • f(2,4)=2

  • f(1,4)=4

而答案為所有數值加總，也就是 12。
