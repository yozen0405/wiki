201 . 獨一無二的區間和(ㄏㄢˋ)

Description

給定正整數 N，判斷是否存在長度為 N 的序列 [a1,a2,…,aN](0≤ai<230) 使得任意非空區間 [al,al+1,…,ar](1≤l≤r≤N) 的位元和(ㄏㄢˋ)和(ㄏㄜˊ) (bitwise AND sum) 是獨一無二的。如果有，請構造任意滿足這個條件的序列。

換句話說，集合

S={al&al+1&⋯&ar∣l,r∈N,1≤l≤r≤N}

的大小必須是 (N+12)。

Input Format

每份測試檔案的第一行（也是唯一一行）會包含恰一個整數 N，代表需要構造的序列長度。

  • 1≤N≤5000

Output Format

對於每筆測試資料，如果可以構造出滿足條件的序列，第一行請輸出 "Yes"（不含引號）；反之輸出"No"（不含引號）。

如果第一行的輸出是為 "Yes"，請輸出第二行 a1,a2,…,aN 代表你所構造出來的序列。兩兩整數之間以半形空白分開。
