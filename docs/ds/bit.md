## BIT 支援操作

query 的區間只能是前綴, 僅能回答可由前綴組合出來的問題

1. 區間求和, 單點加值
2. 單點求和, 區間加值
3. 區間求和, 區間加值



### 區間求和, 單點加值

註 : 其實把加值變化一下可以支援單點改值



### 單點求和, 區間加值

把 bit 的看成是前綴和陣列，區間加值就用差分改兩個點，單點求和時就求 bit[1] + ... + bit[i]

### 區間求和, 區間加值

維護 bit[i] = 4 * a[1] +  3 * a[2] + 2 * a[3] + 1 * a[4]

a[i] + ... + a[j]

= (pre[1] + ... + pre[i]) + ... + (pre[1] + ... + pre[j])
